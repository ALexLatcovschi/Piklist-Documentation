### `'type' => 'colorpicker'`


Create a WordPress colorpicker field.

You can use Piklist field parameters to customize this field.

```
piklist('field', array(
    'type' => 'colorpicker',
    'field' => 'my_color',
    'label' => 'Color Picker'
  ));
```
